# 单项选择测评应用

一个专门针对单项选择的测评应用

## 依赖

- jQuery
- AmazeUI

## 安装

- `git clone <repository-url>`
- `npm install`

## 运行

- `npm start`

## 访问

在浏览器输入`http://localhost:3000`进行访问
